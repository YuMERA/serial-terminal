    Help

    To get started with the Web Serial Terminal:

    • Connect your device to your computer using a USB cable.
    • Click the Connect button.
    • From the list of available serial ports, select the one corresponding to your device.
    • Once connected, you can send and receive data using the terminal interface below.
    • Use the predefined AT command list for quick access to commonly used commands.
    • Customize your terminal experience via the Settings icon.

    This application is designed to make serial communication simple, fast, and accessible directly from your browser.

    Application Features & Notes

    • Web Serial API: Direct browser access to serial ports
    • Auto Scroll: Works only while the buffer is not full (under Max Lines). After that, scroll is always forced.
    • Copy / Save: Only the most recent Max Lines will be copied or saved. Older data is removed from memory.
    • RTS/DTR Control: User-configurable signals with status indicators
    • Pause Output: Pauses live data display. Incoming data is ignored during pause.
    • Timestamp: Adds real-time timestamp (12h or 24h) to each line if enabled.
    • Display Format: Switch between Normal / Hex / JSON views for incoming data.
    • Search and Highlight: Inline search with results count
    • Auto-Clear: When active and connected, clears the terminal after selected time interval.
    • Dark/Light Theme: Switch between themes using the üåô/‚òÄÔ∏è icon. Preference is remembered.
    • Buffer Limit: Configurable in Settings. Determines how many recent lines are stored (min 100).
    • Predefined AT Commands: Quick insert from a popup list of ready-to-use AT commands.
    • Settings Persistence: All options (baud rate, timestamps, themes, etc.) are saved locally.

    These behaviors ensure performance, usability, and stability during continuous serial communication.
